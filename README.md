== ShortcutsWorld

*ShortcutsWorld*[www.shortcutsworld.com] is a RoR Project to show my skills. 


[![Build Status](https://travis-ci.org/cfsbsi/shortcutsworld.svg?branch=master)](https://travis-ci.org/cfsbsi/shortcutsworld)

* Ruby 2.2.3

* Rails 4.2.4

==== Dependencies
* sqllite3
* rpec

=== How to run

 bundle install
 rake db:migrate
 rails server
